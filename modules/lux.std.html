lux.std a LUX module

LUX’s standard module. Here a collection of general-purpose functions are available. This module is also set up in a way that the global environment is accessible through it. This allows you to change the current evironment to it add still have access to the global default environment.

Functions

ls (t)

Prints all key-value pairs of the given table to the standard output.

Parameters:

  • any t

    The table whose field are to be listed.

loadDataFile (path[, loader])

Reads a Lua script as a data file. Makes and returns a table containing everything that was created in the global scope of that script.

Parameters:

  • any path

    The path to the data file.

  • any loader (optional)

    The function used to load the file. Default is Lua’s standard load function.

Returns:

    any

    The table representing the data file.
