Functions

bind (f, ...)
Binds a function to the given arguments. The arguments must be passed in the apropriate order, according to the function's specification.

Parameters

  • f: The function being binded.
  • ...: The binded arguments, in order.

Return value:

A function that, upon being called, does the same as f, but requires only the remaining right-most arguments that were not binded with it.
ls (t)
Prints all key-value pairs of the given table to the standard output.

Parameters

  • t: The table whose field are to be listed.
metabinder (metatable)
Binds the setmetatable function to the given metatable argument.

Parameters

  • metatable: A metatable to be binded to the setmetatable function.

Return value:

A function that takes a table as argument, and sets its metatable to the one given here.
toargs (t, i)
Converts an array to an argument list. Here, an array means the n first elements of a table, indexed from 1 to n. The argument list works exactly like the "..." expression.

Parameters

  • t: The table from where the argument list will be extracted.
  • i: Used for internal tail recursion. No need to worry about it. Defaults to nil, and that is ok.

Return value:

An argument list that works just like a "..." expression.
