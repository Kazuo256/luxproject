Module nova.object

LuaNova's object module. This module is used to create objects from prototypes, through the use of the nova.object:new() method. It also defines a reference to a nil object, which may be acquired with nova.object.nilref().

Functions

inherit (super)
Makes a class module inherit from a table. This function is to be used only when declaring modules, like this:

module ("my.module", nova.object.inherit(some_table))

This essentially makes the module inherit everything indexable from the given table. It also turns the module into an object.

Parameters

  • super: A table from which the module will inherit.
nilref ()
Returns the representation of the nil object.

Return value:

An object reference to the nil object.
