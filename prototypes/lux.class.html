lux.class a LUX prototype

A class-based implementation object oriented programming. Ironically, this is actually a prototype, which means it inherits from lux.prototype, but otherwise provides its own mechanism for OOP.

Be sure to check instance, inherit and super usages.

Functions

class:instance (obj, ...)

Defines how an instance of the class should be constructed.

Parameters:

  • any obj

    The to-be-constructed object

  • any ...

    Any arguments required by the construction

Usage:

      local MyClass = require 'lux.class' :new{}
  function MyClass:instance (obj)
    local a_number = 42
    function obj:show ()
      print(a_number)
    end
  end

class:inherit (another_class)

Makes this class inherit from another. It is necessary to call self:super(obj, …) inside this class' instance definition method.

Parameters:

  • any another_class

    The being inherited from

Usage:

      local class = require 'lux.class'
  local ParentClass = class:new{}
  local ChildClass = class:new{}
  ChildClass:inherit(ParentClass)

class:super (obj, ...)

Calls the parent class' constructor. Should only be called inside this class' instance definition method when it inherits from another class.

Parameters:

  • any obj

    The object being constructed by the child class

  • any ...

    The parent class' constructor parameters

Usage:

      -- After ChildClass inherited ParentClass
  function ChildClass:instance (obj, x, y)
    self:super(obj, x + y) -- parent's constructor parameters
    -- Finish instancing
  end
